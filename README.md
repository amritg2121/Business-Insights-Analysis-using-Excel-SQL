# Retail Business Analysis using Excel & SQL

## Project Overview

This project analyzes retail sales data using Excel and MySQL to generate business insights.

## Tools Used

* Microsoft Excel
* MySQL Workbench

## Excel Skills

* Pivot Tables
* XLOOKUP
* Power Query
* Data Cleaning

## SQL Skills

* GROUP BY
* Aggregate Functions
* ORDER BY
* Window Functions (RANK)

## Analysis Performed

* Regional Sales Analysis
* Category Profit Analysis
* Product Performance Analysis
* Customer Type Analysis

## Dataset

158 records imported and analyzed using MySQL.

## Files Included

* Dataset (CSV)
* SQL Query Scripts
* Query Result Screenshots
* Excel Analysis Workbook
